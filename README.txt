# Site Automatiza (Landing Page)
Estrutura separada em HTML, CSS e JS.

## Arquivos
- `index.html` — marcação da página
- `assets/styles.css` — estilos
- `assets/app.js` — interações (FAQ, formulário)
- `assets/og-image.jpg` e `assets/favicon.ico` — substitua pelos seus arquivos de imagem

## Como rodar localmente
- VS Code com extensão **Live Server** → clique com direito no `index.html` → *Open with Live Server*.
ou
- Terminal: `python -m http.server` e acesse http://localhost:8000

## Publicação rápida
- Netlify / Vercel / GitHub Pages: faça deploy da pasta `site-automatiza/` inteira.
